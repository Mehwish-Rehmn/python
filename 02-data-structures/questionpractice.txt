### lists

1. create an empty list called fruits  
2. add "apple", "banana", "cherry" to the list fruits  
3. print the second item of the list fruits  
4. change the first item of fruits to "mango"  
5. add "orange" at the end of the list fruits  
6. remove "banana" from the list fruits  
7. print the length of the list fruits  
8. sort the list fruits in alphabetical order  
9. create a new list called numbers = [5, 12, 3, 8, 1] and print only numbers greater than 4  
10. make a list of 5 city names, then reverse it and print the result  

### tuples

4. create a tuple with only one item: number 100

7. check if "sunday" is inside the tuple days
