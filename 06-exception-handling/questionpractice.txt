Questions practice

1. Create a CSV file named bootcamp_students.csv with columns:
name, python_score, ml_score, devops_score.

2. Write a program that:
Reads the CSV file safely.
Converts scores to integers.
Calculates the average score of each student.
Prints student name, average, and grade.

3. Assign grades using:
90+ - A
75–89 - B
60–74 - C
Below 60 - D

4. Find and print the topper (student with highest average).

5. Handle the following errors:
File not found
Invalid number format in scores
Missing column in CSV

6. Modify the program to:
Add a new student record
Save updated data back to CSV

7. Bonus:
Save the topper’s data in JSON format.
Display grade distribution count (how many A, B, C, D).
